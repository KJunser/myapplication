### TeamTogether

# 개요
1인 가구의 증가와 배달음식의 수요 증가로 인해
혼자 먹기엔 부담되는 최소금액과 점점 올라가는 배달료의 부담을
주변사람들과 함께 '배달 공동구매'를 하여 음식을 공유했으면 좋겠다 라는 생각으로 만들었습니다.
